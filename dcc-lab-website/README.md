# DCC Lab – PHD1 MVP

**How to use**
- Place demo videos in `assets/videos/`:
  - phd1-phd.mp4
  - phd1-bc.mp4
  - phd1-md.mp4
  - phd1-fhc.mp4
  - phd1-contemporary.mp4

- Put logo at `assets/images/logo.png` (optional).
- Add poster images for videos (optional) at `assets/images/`.

**Edit**
- Update `js/main.js` with your WhatsApp phone number (international format, no +).
- Replace `REPLACE_WITH_EMAIL` in contact.html with your contact email.
- Optionally set Gumroad and Telegram links in `js/main.js`.

**Preview locally**
- Double-click `index.html` to open in your browser.

**Deploy to GitHub Pages**
1. Create GitHub repo (public) named `dcc-lab-website`.
2. Upload files via "Add file" → "Upload files".
3. Commit changes.
4. Settings → Pages → Branch: main (root) → Save.
5. After a few minutes visit: `https://your-username.github.io/dcc-lab-website/`.
